# UIT-HWDB dataset

## Statistic of the dataset

| Dataset | Total |
|:---|:---|
| UIT-HWDB-word | 110,745|
| UIT-HWDB-line | 7,273 |
| UIT-HWDB-paragraph | 1,144 |
| UIT-HWDB-word-syn | 307,734 |
| UIT-HWDB-line-syn | 36,320 |
